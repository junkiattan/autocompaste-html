Welcome!

The purpose of this experiment is to determine the performance of 2 copy techniques, the normal Copy-Paste technique using Ctrl-C and Ctrl-V, and a new technique called AutoComPaste, which allows a user to copy over any content by suggesting a list of completions as the user types.

A participant number should have been passed to you. This number would aid us in tracking your progress and collect relevant experimental data.

Please write your participant number ID given by your experimenter. Participant IDs should only consist of alphanumeric characters.
